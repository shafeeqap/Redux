Formik
------
Formik is a library that helps with handling forms in React. 
It provides a set of tools to manage form state, handle form submissions, and manage validation.
Formik aims to reduce the amount of boilerplate code required for building forms, 
making it easier to manage complex form logic.

Yup
---
Yup is a JavaScript schema builder for value parsing and validation. 
It is often used with Formik to handle form validation. 
Yup allows you to define a schema for your form fields and validate the form values against this schema.
